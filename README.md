- Ball bounces off paddles and top/bottom walls
- Score increases when the opponent misses

## How to Run
### Option A: Run from Visual Studio (Windows)
1. Open the project in **Visual Studio**
2. Make sure you have your graphics library set up (ex: **raylib**)
3. Press **Start (F5)** or **Ctrl + F5**
